# NeciAI - Intelligent Financial Data Analysis Platform

## Overview
NeciAI is a cloud-hosted, AI-powered financial data analysis platform built with C# (ASP.NET Core 8), Python (FastAPI), and React. It enables business professionals to upload financial data, receive AI-generated forecasts, detect anomalies, and export professional financial reports.

## Tech Stack
- Backend API: C# ASP.NET Core 8
- AI Microservice: Python 3.12 + FastAPI
- Frontend: React 18 + Tailwind CSS + Chart.js
- Database: PostgreSQL (Neon)
- Auth: JWT + ASP.NET Identity
- Deployment: Vercel + Render + Docker
